ULA using Shenzhen I/O

I designed a custom 4-bit ALU (Arithmetic Logic Unit) in the game Shenzhen I/O using multiple microcontrollers, each limited to two input pins: P0 and P1. The ALU supports two operations: addition and subtraction of 2-bit binary numbers. Inputs are provided via switches, and two additional switches are used to select the operation mode. Each controller is responsible for a specific part of the operation, and one dedicated controller defines which operation to execute. The results are passed through outputs X0 to X3 and are displayed on LEDs connected to another controller. No built-in registers like A, B, or CONTROL were used; instead, I used only basic inter-controller connections and logic to simulate real ALU behavior in a constrained environment.
